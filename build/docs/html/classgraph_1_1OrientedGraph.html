inline

Получить множество вершин, с которыми соединена указанная вершина.

Аргументы
idНомер вершины.

Функция возвращает множество вершин в виде хеш-таблицы, с которыми соединена вершина id, то есть множество таких вершин V, что ребро (id, V) присутствует в графе. Если указанной вершини в графе нет, то функция выбрасывает исключение std::out_of_range.
